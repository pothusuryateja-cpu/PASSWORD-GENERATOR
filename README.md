Password Generator — Python

This project is a simple Python-based password generator that creates strong and secure passwords using a mix of uppercase letters, lowercase letters, numbers, and special characters. The user can specify the desired password length, and the program generates a randomized password accordingly.

Features

Generates strong random passwords

Supports letters, digits, and special characters

Customizable password length

Beginner-friendly and easy to modify

Works in any terminal or IDE

Tech Stack

Python 3

random module

string module

Installation and Usage

Make sure Python is installed on your system.

Run the program from your terminal or IDE.

Enter the desired password length when prompted.

The program will display a randomly generated password.
